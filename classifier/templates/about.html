Sobre o Projeto

O projeto

Este site é uma demonstração dos resultados obtidos no projeto "Detecção Automática de Notícias Falsas para o Português", financiado pelo Programa Institucional de Bolsas de Iniciação Científica (PIBIC) do CNPq. O projeto visava estudar métodos para a detecção automática de notícias falsas utilizando Processamento de Linguagem Natural (PLN) e Aprendizado de Máquina (AM).

Financiamento e Apoio

Este projeto foi desenvolvido no Instituto de Ciências Matemáticas e de Computação da Universidade de São Paulo (ICMC-USP), no campus São Carlos. O projeto foi financiado pelo Programa Institucional de Bolsas de Iniciação Científica (PIBIC) do CNPq e foi desenvolvido com o apoio do Núcleo Interinstitucional de Linguística Computacional (NILC)..
