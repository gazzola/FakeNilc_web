Sobre o Projeto

Como funciona?

Ao receber um texto, o sistema aplica métodos para extrair atributos linguísticos desse texto e os utiliza em um modelo de aprendizado de máquina, que classifica a notícia como verdadeira ou falsa. Os modelos disponibilizados foram treinados com o córpus Fake.Br.

Estão disponíveis dois modelos de detecção: "Frequência de Palavras" e "Classes Gramaticais". O modelo de frequência de palavras utiliza uma representação Bag of Words do texto, onde a presença ou ausência de uma palavra é marcada como 0 ou 1 em um vetor com 10395 posições (relativas às palavras mais importantes do córpus). O modelo de classes gramaticais calcula a porcentagem de palavras pertencentes à cada classe sintática, de acordo com o tagger nlpnet.

Os atributos extraídos do texto são aplicados em um classificador Support Vector Machine, que infere a classe da notícia (verdadeira ou falsa). Nos testes realizados, o sistema obteve cerca de 88% de acerto (acurácia geral). Porém, sendo esse sistema apenas uma prova de conceito, não é recomendado o uso do FakeCheck como única fonte de verificação de notícias. Busque fontes confiáveis para todas as notícias que você ler/compartilhar na web!

O projeto

Este site é uma demonstração dos resultados obtidos no projeto "Detecção Automática de Notícias Falsas para o Português", financiado pelo Programa Institucional de Bolsas de Iniciação Científica (PIBIC) do CNPq. O projeto visava estudar métodos para a detecção automática de notícias falsas utilizando Processamento de Linguagem Natural (PLN) e Aprendizado de Máquina (AM).
